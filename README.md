first install pnpm globally
```bash
npm install -g pnpm
```

then install the dependencies
```bash
pnpm install
```

then make sure you have the composer installed and run
```bash
composer install
```

then run the migrations
```bash
php artisan migrate
```


then run serve
```bash
php artisan serve
```

then run the frontend
```bash
pnpm dev
```

then open the browser and go to
```bash
http://127.0.0.1:8000/
```
